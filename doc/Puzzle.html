class Eq p => Puzzle p where
Puzzle is a class of puzzles solved by brute-force backtracking. The class requires solved and choices and provides solutions and solve.
Methods
solved :: p -> Bool
solved puzzle is true if the puzzle is solved.
choices :: p -> [p]
choices puzzle returns a list of possible new puzzles if the puzzle is not solved.
solve :: p -> p
solve puzzle returns the first solution of a puzzle, if any.
solutions :: p -> [p]
solutions puzzle returns a list of all solutions for a puzzle.
